# Todo Time Square ⏰✅

**随性选择 · 深度专注**

</div>

<div align="center">

专为追求灵活性与自发性的创造者（如MBTI中的P型特质）设计的效率伴侣：

“融合开放式任务管理与可定制番茄工作法，让您从自由池中即时挑选任务，一键进入无干扰的心流状态。”

</div>

<div align="center">

## 设计哲学

</div>

<div align="center">

我们相信真正的生产力源于自由而非束缚。传统日程工具要求严格规划，而我们将宏观目标拆解为独立、可随时执行的微任务。您无需预先安排全天计划，只需从开放池中选取当下最想完成的任务，立即开启一段专注旅程。每次冲刺后，系统将自动记录进度并提供直观反馈，让持续进步成为自然习惯。

</div>

<div align="center">

## 主要特性

</div>

<div align="center">

- **自由任务池**：无顺序限制，随时添加、浏览或移除，保持思维轻盈。
- **智能番茄钟**：自定义专注与休息时长，从任务池一键启动专属冲刺。
- **沉浸式界面**：极简设计，自动屏蔽干扰，实时可视化进度与时间流逝。
- **成就系统**：每次专注后自动标记进度，累积数据生成个人效率洞察。
- **全平台适配**：基于Flutter构建，提供流畅一致的跨设备体验。

</div>

<div align="center">

## 为何选择此方法？

</div>

<div align="center">

- **决策极简化**：将"今天做什么"的庞大压力转化为"现在做这件"的轻松选择。
- **灵活变专注**：顺应自发思维模式，将跳跃性灵感转化为可执行的短周期产出。
- **正向循环激励**：即时视觉反馈与数据累积，潜移默化构建持续专注的成就感。

</div>

<div align="center">

## 快速入门

</div>

<div align="center">

1.  获取项目代码：
    ```bash
    git clone https://github.com/ChenXu233/ToDoTimeSquare.git
    cd ToDoTimeSquare
    ```

2.  安装依赖并准备本地化资源：
    ```bash
    flutter pub get
    flutter gen-l10n
    ```

3.  运行应用程序：
    ```bash
    flutter run
    ```
    （可指定平台，例如：`flutter run -d windows`）

</div>
